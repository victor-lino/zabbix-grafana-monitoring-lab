## Environment Description

---

## How to Use

1. Download the .ova or .ovf + .vmdk files
2. Open VMware Workstation or VMware Player
3. Go to File → Open
4. Select the downloaded VM file
5. Start the virtual machine
6. Verify network connectivity and agent status
